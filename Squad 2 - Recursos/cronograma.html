Descrição das Disciplinas

POO (Programação Orientada a Objetos): Implementação prática de herança e polimorfismo em um sistema simples usando Java.

IoT (Internet das Coisas): Configuração e programação básica de sensores para coleta de dados via Arduino.

Matemática: Resolução de exercícios sobre lógica proposicional e conjuntos para fortalecer o raciocínio lógico.

Inglês Técnico: Leitura e tradução de artigos técnicos relacionados a desenvolvimento de software.

Front-End: Construção do layout inicial de uma página web com HTML e aplicação de estilos básicos em CSS.

Sistemas Operacionais: Estudo dos conceitos de gerenciamento de processos e escalonamento de tarefas.

Metodologias: Simulação de um fluxo ágil usando Scrum, com definição de sprints e papéis.

Descanso: Pausa para alongamentos e relaxamento, para garantir maior concentração no restante do dia.
